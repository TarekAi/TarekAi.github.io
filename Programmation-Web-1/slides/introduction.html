Qu’est-ce que le Web ?

Le Web est un système d’information basé sur :

  • Le protocole HTTP
  • Les navigateurs (clients)
  • Les serveurs web
  • Les ressources (pages HTML, CSS, images, etc.)

Histoire du Web

Inventé en 1989 par Tim Berners-Lee au CERN.

Le Web repose sur les standards du W3C et s’est rapidement imposé comme un outil universel de communication.
